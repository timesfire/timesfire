Mac 使用常见问题汇集

1、Mac系统如何隐藏文件或文件夹？

在 Mac 文件的文件名前添加 . 即可使文件隐藏

2、Mac系统如何查看隐藏文件？

终端里面能用命令来控制:

显示Mac隐藏文件的命令：defaults write com.apple.finder AppleShowAllFiles YES
隐藏Mac隐藏文件的命令：defaults write com.apple.finder AppleShowAllFiles NO

命令运行后需要按 Command Option Esc 选中 Finder 然后点一下 重新开启 就 Ok 了

3、Mac系统如何像windows那样创建桌面快捷方式？

在Finder中找到需要创建桌面快捷方式的文件或文件夹，同时按住alt+command，将文件用鼠标拖拽到桌面，此时鼠标会变成箭头，保持alt+command的按下状态，松开鼠标即可。

4、Mac系统中Finder边栏的文件夹名称之前是中文，不知怎么变英文了，怎样改回中文？

在相应文件夹下新建一个名称为.localized的空文件即可恢复
