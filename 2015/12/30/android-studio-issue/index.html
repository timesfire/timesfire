Android Stuido 使用过程中遇到的问题

1、新建Java library 项目编译时出错：Gradle Error:Cause: peer not authenticated

将build.gradle文件里的：

repositories {
     jcenter()
}

替换为：

repositories {
     jcenter
         {
           url 'http://jcenter.bintray.com'
         }
}
